# Brainfuck 解释器使用示例

## Brainfuck 8个基本命令

| 命令 | 说明 |
|------|------|
| `>` | 数据指针右移 |
| `<` | 数据指针左移 |
| `+` | 当前单元值加1 |
| `-` | 当前单元值减1 |
| `.` | 输出当前单元值的ASCII字符 |
| `,` | 输入一个字符到当前单元 |
| `[` | 如果当前单元值为0，跳转到匹配的`]` |
| `]` | 如果当前单元值非0，跳转到匹配的`[` |

## 使用方法

### 方法1: 直接执行代码
```bash
./brainfuck '代码'
```

### 方法2: 从文件执行
```bash
./brainfuck filename.bf
```

## 示例1: Hello World
```bash
./brainfuck '++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.'
```

输出: `Hello World!`

## 示例2: 打印"A"
```bash
./brainfuck '++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.>.'
```

输出: `ABC`

## 示例3: 简单循环 - 打印"Hi"
```bash
./brainfuck '+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++.>++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++.'
```

输出: `Hi`

## 示例4: 乘法 (3*4=12)
此程序计算两个数的乘积:
```bash
./brainfuck '+++>++++[<[>+<-]>[<+>-]>-]<.'
```

## 示例5: 从文件读取代码
创建 `hello.bf`:
```
++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.
```

然后执行:
```bash
./brainfuck hello.bf
```
